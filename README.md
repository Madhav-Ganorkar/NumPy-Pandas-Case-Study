Perform basic sales data analysis using NumPy and Pandas.

## 🏗️ Setup
- Python 3.x
- pandas
- numpy

---

## ▶️ How to Run

Compute region-wise total sales

Average sales per product

Identify highest and lowest selling products

Statistical summary using NumPy
